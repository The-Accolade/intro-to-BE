What does the backend mean?
"Backend": server-side that focuses on databases, scripting and website architecture. it is basically the part of the website that users don't see.

Frontend runs on the browser, the backend of the website runs on a server.

     MERN (MongoDB, ExpressJS, ReactJS and NodeJS), 
     MEAN(MongoDB, ExpressJS, Angular and NodeJS), 
     MEVN (MongoDB, ExpressJS, VueJS and NodeJS),

What are APIs? 
Application Programming Interface. They are basically a way for different software systems to communicate with each other, allowing for integration of functionalities. The API creator creates a set of rules and protocols for building and interacting with his own software, defining the methods, data and formats that the applications can use to communicate with each other.
